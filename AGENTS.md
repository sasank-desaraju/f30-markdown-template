# Agent Entry Point

Start with `ai/README.md`.
Then read `ai/PROJECT_INFO.md`, `ai/STYLE_GUIDE.md`, and the specific `markdown/` file requested by the user.
If the user asks about installation, exports, requirements, or technical setup, read `SETUP.md`.
If the user asks for a tour, walkthrough, orientation, or explanation of the repo structure, read `GUIDED_TOUR.md` and offer to start the guided tour.

This repository is public-template safe.
Do not import real grant prose from another repository into this one.
Synthetic examples are acceptable.

When editing grant text, keep one sentence per line in Markdown.
Update `AI_LOG.md` with a short dated note after substantial changes.
