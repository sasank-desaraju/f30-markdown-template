# AI Context Entry Point

This directory is the canonical location for project context and writing guidance for AI assistants working in this repository.
Start here when you need to understand the synthetic F30 narrative, aims, terminology, file layout, and preferred writing style.

## What To Read

- `ai/PROJECT_INFO.md` contains project overview, research focus, aims, terminology, mentorship team, and document map.
- `ai/STYLE_GUIDE.md` contains grant-specific writing principles, review checklist, and formatting conventions.
- `ai/NAVIGATION_AND_PERMISSIONS.md` contains terminal navigation guidance and default permissions.
- `docs/workflows/` contains human-facing workflow notes for common grant-writing tasks.

## Working Assumptions

- Markdown files in `markdown/` are the source of truth.
- Generated Word and PDF files are review artifacts.
- Public template content must remain synthetic.
- Real grant content, private preliminary data, PHI, and confidential letters should not be added to this repository.
