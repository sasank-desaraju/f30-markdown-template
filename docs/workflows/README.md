# Workflows

These guides describe common workflows for a Markdown-first F30 application.
They are intentionally practical and assume collaborators may still prefer Word files for review.

- `writing_loop.md`: daily writing and AI revision loop.
- `word_review_export.md`: exporting sections for mentor markup.
- `feedback_triage.md`: turning scattered comments into actionable edits.
- `submission_export.md`: producing separate attachment PDFs.
- `citation_workflow.md`: keeping references sane.
