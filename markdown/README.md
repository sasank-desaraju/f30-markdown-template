# QMD Source Files

This folder contains the Markdown and Quarto source files for the synthetic F30 application.
Edit these files rather than generated Word or PDF exports.

Recommended workflow:

1. Draft and revise in Markdown.
2. Export a section to Word for mentor comments.
3. Put marked-up review copies in `feedback/`.
4. Reconcile accepted changes back into Markdown.
5. Render the full draft from `draft.qmd`.
