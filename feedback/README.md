# Feedback

Drop marked-up Word files, reviewer comments, mentor emails, mock study-section notes, and AI critique here.
When feedback is actionable, summarize it in a Markdown file so humans and agents can triage it.

Suggested naming:

- `YYYY-MM-DD_mentor_aims_feedback.md`
- `YYYY-MM-DD_mock_study_section_notes.md`
- `YYYY-MM-DD_word_markup_summary.md`
